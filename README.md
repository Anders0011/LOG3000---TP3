Objectif:  
L'objectif de ce projet est de créer une calculatrice web simple en utilisant la biblothèque Flask de Python.


Prérequis d'installation: 
Il faut avoir installé python et pip installés sur son ordinateur afin de pouvoir lancer le projet. 

Instructions d'installation: 
1. Il faut cloner le dépot dans le répertoire de son choix.
2. Créer un environnement virtuel dans le projet.
3. Installer les dépendances nécessaires
4. Lancer l'application
